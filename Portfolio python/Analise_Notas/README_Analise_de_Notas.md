# Análise de Notas com Listas

O script coleta 3 notas do usuário, armazena em uma lista e exibe:

- Todas as notas digitadas
- Maior nota
- Menor nota
- Soma das notas
- Média das notas

## Como usar

1. Execute o script.
2. Digite 3 notas quando solicitado.

## Exemplo de saída

```
Notas [7.0, 9.0, 6.5]
A maior nota é: 9.0
A menor nota é: 6.5
A soma de todas as notas é: 22.5
A média é: 7.5
```
